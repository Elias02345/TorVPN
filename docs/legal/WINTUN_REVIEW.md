# Wintun Review Gate

Windows support may use Wintun only after legal and packaging review confirms the integration model is compatible with TorTunnel's GPL-2.0-or-later license.

Planned constraints:

- Use official, unmodified, signed Wintun components when possible.
- Do not rename or modify Wintun driver artifacts.
- Keep Wintun provenance, hashes, and license text in release artifacts.
- If compatibility cannot be confirmed, Windows must switch to a different TUN/WFP strategy before stable release.
