# Stable Release Blockers

TorTunnel cannot publish a stable build until all blockers are closed:

- Android, Linux, and Windows native adapters are implemented and signed.
- Strict Mode leak matrix passes on real hardware or reproducible VMs.
- External security/privacy audit is complete.
- All High and Medium audit findings are fixed and re-verified.
- Wintun/legal packaging review is complete.
- Reproducible Tor, obfs4proxy, and Snowflake build provenance is published.
- F-Droid, GitHub, Flatpak, AppImage, DEB, RPM, and MSI artifacts are signed.
